# hackathon-neurafleet

2025 November Hackathon at Zayed University 

Logistics Team 

## Idea name: Neura-Fleet

A hybrid (drone + bike) system allowing faster and efficient deliveries across the UAE. 
Drone monitoring contains the web-app to be used to manage and monitor all deployed drones and bikes.
Sustainability Analysis contains the web-app to be used to show key KPI and analysis in reducing carbon emission.
